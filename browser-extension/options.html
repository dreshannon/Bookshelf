Bookshelf Commit — Options

Configure which repo to commit to and the personal access token the extension uses. The token is stored in the browser's synced storage and never leaves your machine except when talking to api.github.com.

Recommended PAT setup: fine-grained token, scoped to just this repo, with Contents: Read and write permission.
  1. Open GitHub → Settings → Developer settings → Fine-grained tokens.
  2. New token, give it a recognisable name.
  3. Repository access → Only select repositories → pick your Bookshelf repo.
  4. Permissions → Repository permissions → Contents: Read and write.
  5. Copy the token and paste it above.
